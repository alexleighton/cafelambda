# Cafe Lambda

A just-for-fun implementation of a Lambda Calculus interpreter, in Java.


## Setup

Known good setup:
  * Java JDK: 11.0.1 (Oracle Corporation 11.0.1+13-LTS)
  * Gradle: 4.10.2 (can be installed via [Homebrew](https://brew.sh/))
  * IntelliJ: 2018.3


## Test

    $ gradle test


## Run

    $ gradle run
